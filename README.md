# Info

This is a customizable autoclicker offering tons of settings and features, with support for both **Linux** and **Windows**.

---


# Features

- Custom themes
- Screen edge detection (stop when touching edge)
- Click only on a specific window
- Good performance
- Randomize click delay
- Hide from taskbar
